# Selenium Test Automation Project

This project contains automated UI tests for the Saucedemo login form.
Includes:
- Selenium WebDriver
- JUnit 5
- AssertJ
- SLF4J
- Data Provider
- Parallel execution (JUnit)
- Firefox + Edge browsers

## Test cases
UC-1: Empty credentials  
UC-2: Missing password  
UC-3: Valid credentials

Run tests:
```
mvn test
```
